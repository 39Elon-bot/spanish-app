# ¡Hablemos! 🇪🇸 — 半年でスペイン語会話アプリ

スペイン語ほぼ初心者から、**半年でネイティブと日常会話ができるレベル**(ネイティブ小学校高学年相当の会話力)を目指す学習アプリです。

- **単語・リスニング・スピーキング特化**(読み書きはやりません)
- 全432項目(単語288 + フレーズ144)に **スペイン語 / 英語 / 日本語** の3言語表記
- 文法解説なし。「使えるフレーズを、聞いて、言える」ことだけに集中

## カリキュラム(6ヶ月 = 6フェーズ × 4ユニット)

| 月 | フェーズ | 内容 |
|---|---|---|
| 1ヶ月目 | サバイバル基礎 | あいさつ / 自己紹介 / 数字 / 家族 |
| 2ヶ月目 | 日常の基本 | 食べ物 / レストラン / 時間 / 毎日の動作 |
| 3ヶ月目 | 街へ出よう | 道案内 / 買い物 / 乗り物 / 天気 |
| 4ヶ月目 | 気持ちを伝える | 感情・体調 / 趣味 / お願い / 過去の話 |
| 5ヶ月目 | 会話を広げる | 予定・未来 / 意見 / 電話 / 緊急時 |
| 6ヶ月目 | ネイティブに近づく | 相づち / スラング / 話をつなげる / 卒業 |

各ユニットは **📖 学ぶ → 🎧 聞く → 🗣 話す → 🏆 テスト** の4ステップ。
テストに合格すると次のユニットが解放されます(1日1ステップ、約5〜10分でOK)。

## 🎧 ¡Escucha! — 聞き流しサブアプリ(`listen/`)

本編とは別に、**聞き流し専用アプリ** が `listen/` にあります(ホーム画面のリンク、または `/listen/` を直接開く)。

- **単語帳**: 本編に収録していない日常頻出 **約1,900語**(名詞は冠詞付き)を es → en → ja の順で自動読み上げ
- **フレーズ集**: ネイティブがよく使う短いフレーズ **384本**
- **1サイクル ≈ 15分**(単語120語 × 16サイクル / フレーズ96本 × 4サイクル)
- サイクルリピート / 通し再生 / 全体リピート、**自動レジューム**(続きから再生)
- ロック画面のメディアコントロール対応。ロック中に音が止まる端末向けに **🌙黒画面モード**(画面真っ黒+スリープ防止)あり — ランニングの聞き流しに

## 続けたくなる仕組み

- 🔥 連続学習日数(ストリーク)
- ⭐ XPとレベル(入門者 → ネイティブ小学生レベル)
- 🔁 忘れかけたタイミングで出題される「今日の復習」(間隔反復)
- 😅 間違えた言葉だけの「ニガテ克服クイズ」
- 🎉 ユニットクリアで紙吹雪

## 使い方

### パソコンで試す

```bash
cd ~/Desktop/Spanish
node serve.mjs
# → http://localhost:8765 を開く
```

### スマホで使う(推奨: GitHub Pages)

音声認識(🗣話すモード)は **https** が必要なので、無料のGitHub Pagesが一番簡単です。

1. GitHubで新しいリポジトリを作る(例: `spanish-app`)
2. このフォルダの中身をアップロード
3. リポジトリの Settings → Pages → Branch を `main` にして保存
4. `https://あなたのID.github.io/spanish-app/` をスマホで開く
5. **ホーム画面に追加**(iPhone: 共有 → ホーム画面に追加 / Android: メニュー → アプリをインストール)すると、アプリとして全画面で使えます。オフラインでも動きます

### 対応ブラウザ

- 音声読み上げ(TTS): ほぼ全ブラウザOK
- 音声認識(話すモード): **Chrome / Safari** 推奨。使えない環境では自動的に「自己チェック方式」に切り替わります
- 設定タブで声の種類と話す速さを変えられます(iPhoneなら「Mónica」、高品質音声を設定アプリからダウンロードするとさらに自然になります)

## ファイル構成

| ファイル | 役割 |
|---|---|
| `index.html` | アプリ本体のページ |
| `data.js` | カリキュラム全データ(単語・フレーズの3言語対訳) |
| `app.js` | アプリのロジック(画面・音声・進捗・ゲーム要素) |
| `styles.css` | デザイン(モバイルファースト) |
| `manifest.webmanifest` / `sw.js` | ホーム画面追加・オフライン対応(PWA) |
| `serve.mjs` | パソコンで試すための簡易サーバー |

進捗はブラウザの `localStorage` に保存されます(サーバー不要・アカウント不要)。
